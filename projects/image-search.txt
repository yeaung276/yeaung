b:Tc8a,<h2>Neural Style Transfer</h2>
<h3>Introduction</h3>
<p>This is Neural Style Transfer implementation using vgg-net-deep-19 conv-net and Gram metrix. The hypermeters in here are used from Coursera-coursed(Convolutional Neural Network). You can expreiment with different parameter in <a href="https://colab.research.google.com/github/yeaung276/NeuralStyleTransfer/blob/master/NeuralStyleTransfer.ipynb">this colab notebook</a>.</p>
<h3>How to run the project</h3>
<p>You need to have docker installed in your machine. Run
<code>docker-compose up</code>
to start the service and go to
<code>localhost:8000</code> to interact with the app. For manual running <code>core.NST</code>, <code>main.py</code> file is provided. You will need to download VGG weight metrix from <a href="https://www.vlfeat.org/matconvnet/models/imagenet-vgg-verydeep-19.mat">here</a>.

useful links:</p>
<ul>
<li><code>localhost:8000</code> for interacting with app</li>
<li><code>localhost:8001</code> for interacting with redis</li>
<li><code>localhost:8000/docs</code> for interactive api documentation</li>
</ul>
<h3>Project Architecture</h3>
<p>This app use</p>
<ul>
<li><code>redis</code> for saving app data and outputs,</li>
<li><code>fastApi</code> for inferencit ML model</li>
<li><code>websocket</code> for realtime notification</li>
<li><code>asyncio</code> and <code>multiprocessing</code> for running the NST job in the background, and</li>
<li><code>docker</code> for containerization the app</li>
</ul>
<p>In this app, if content image and style image are not same size, procrocessor will center crop the images
Enjoy :D</p>
